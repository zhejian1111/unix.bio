# 一、基础使用

## 1、窗口

1）鼠标移动到窗口的四个角，出现十字的时候拖动就可以分割出多个窗口

2）点击十字拖动一个窗口到另一个窗口，黑色的窗口会被吞掉，完成合并



## 2、常用快捷键

### 1）面板的组件

> 按下 <span style={{background: '#f3a694'}}>&nbsp;t&nbsp;</span> 键可以隐藏 **左侧工具栏**

> 按下  <span style={{background: '#f3a694'}}>&nbsp;n&nbsp;</span>  键可以打开 **右侧工具栏**，右侧工具栏实用，插件面板也会出现在这里



### 2）视图的切换

> 按下  <span style={{background: '#f3a694'}}>&nbsp;ctrl + alt + q&nbsp;</span>  切换到 **三视图模式**

> 小键盘上的  <span style={{background: '#f3a694'}}>&nbsp;7、1、3&nbsp;</span>  分别对应着 **正交顶视图，正视图，右视图**

> 在正交视图下，按小键盘的  <span style={{background: '#f3a694'}}>&nbsp;9&nbsp;</span>  ，可以 **翻转视图** 

> 或者直接按住  <span style={{background: '#f3a694'}}>&nbsp;ctrl + 7/1/3&nbsp;</span>  调出 **底、后、左视图**

> 按着  <span style={{background: '#f3a694'}}>&nbsp;shift  + 鼠标中键&nbsp;</span>  可以 **平移视图**



### 3）移动对象

> 使用快捷按钮中的移动按钮进行移动

> 或者按下  <span style={{background: '#f3a694'}}>&nbsp;g&nbsp;</span>  进行 **自由移动**

> 按下  <span style={{background: '#f3a694'}}>&nbsp;g&nbsp;</span>  之后 按下  <span style={{background: '#f3a694'}}>&nbsp;x / y / z&nbsp;</span>  键，可以让物体在 **特定轴** 上移动

> 按下  <span style={{background: '#f3a694'}}>&nbsp;g&nbsp;</span>  之后，按下 <span style={{background: '#f3a694'}}>&nbsp;shift + x / y / z&nbsp;</span> ，可以在 **排除了特定轴的平面** 移动

> 按下  <span style={{background: '#f3a694'}}>&nbsp;g&nbsp;</span>  之后，按住  <span style={{background: '#f3a694'}}>&nbsp;鼠标中键&nbsp;</span>  不放，会出现 **白色虚线** ，将 **白色虚线** 靠近需要移动到的轴之后松开，就可以沿着改轴移动物体位置



### 4）旋转对象

> 按下  <span style={{background: '#f3a694'}}>&nbsp;r&nbsp;</span>  键后，可以让物体在当前的视角下旋转，再此按下  <span style={{background: '#f3a694'}}>&nbsp;r&nbsp;</span>  键，可以让物体没有深度限制的 **自由旋转**

> 与移动对象类似 ， 按下  <span style={{background: '#f3a694'}}>&nbsp;r&nbsp;</span>  键后，按下  <span style={{background: '#f3a694'}}>&nbsp;x / y / z&nbsp;</span>  键，可以让物体在 **特定轴** 上旋转



### 5）缩放对象

> 按下  <span style={{background: '#f3a694'}}>&nbsp;s&nbsp;</span>  键，可以控制物体的 **大小**

> 按下  <span style={{background: '#f3a694'}}>&nbsp;s&nbsp;</span>  键，在 按下按下  <span style={{background: '#f3a694'}}>&nbsp;x / y / z&nbsp;</span>  键，可以让物体在 **特定轴** 缩放



### 6）3d游标

> 按住  <span style={{background: '#f3a694'}}>&nbsp;shift + 鼠标右键&nbsp;</span> ，或者 左侧工具栏的  <span style={{background: '#f3a694'}}>&nbsp;游标按钮&nbsp;</span> 可以设定游标的位置，游标的位置就是 **新增物体时候出现的位置**

> 按下  <span style={{background: '#f3a694'}}>&nbsp;shift + a&nbsp;</span> 可以 **新增物体**



### 7）在物体表面新增物体，实现表面吸附

> 点击左侧工具栏的  <span style={{background: '#f3a694'}}>&nbsp;游标&nbsp;</span>  按钮，在选择上方工具栏中的  <span style={{background: '#f3a694'}}>&nbsp;坐标系&nbsp;</span> ，选择  <span style={{background: '#f3a694'}}>&nbsp;几何数据&nbsp;</span> ，在点击物体，就可以让**游标出现在物体上，并且旋转是与物体保持一致 **

>此时再新增物体，物体会出现在 **游标位置** ，点击左下角  <span style={{background: '#f3a694'}}>&nbsp;添加 XXX&nbsp;</span>  按钮， 对齐选项选择 3D游标 ，就可以 **与原物体旋转一致**，实现吸附在表面 

**或者更简单的办法**

> 按住左侧工具栏中的  <span style={{background: '#f3a694'}}>&nbsp;新增立方体&nbsp;</span> ，<span style={{background: '#f3a694'}}>&nbsp;往右拖动&nbsp;</span> ，会出现添加物体的形状，松开鼠标，就可以在原来物体的表面 <span style={{background: '#f3a694'}}>&nbsp;框选出区域&nbsp;</span> ，就是新物体的大小，之后  <span style={{background: '#f3a694'}}>&nbsp;往外拖动&nbsp;</span> ，添加成功，但是这个方法只能 **添加特定的 5种立方体**

**而 3D 游标 可以添加无限制形状的物体**



### 8）物体B绕着物体A旋转

> 选中物体A，按下  <span style={{background: '#f3a694'}}>&nbsp;shift + s&nbsp;</span>  在饼盘中选择  <span style={{background: '#f3a694'}}>&nbsp;游标 -> 选中项&nbsp;</span> ，就可以将游标设置在物体A中心

接下来有两个方法

方法1、选中物体B，在上方菜单栏的  <span style={{background: '#f3a694'}}>&nbsp;物体&nbsp;</span>  选项中， 设置原点 ，选中  <span style={{background: '#f3a694'}}>&nbsp;原点 -> 3D游标&nbsp;</span> ，之后按下  <span style={{background: '#f3a694'}}>&nbsp;r&nbsp;</span>  旋转，就是**绕着物体A 旋转**

方法2、选中物体B，在上面中间的菜单栏  <span style={{background: '#f3a694'}}>&nbsp;变换轴心点&nbsp;</span>  中选择  <span style={{background: '#f3a694'}}>&nbsp;3D游标&nbsp;</span> ，也能绕着A旋转

旋转物体A，物体B也会跟着旋转

### 4）万向坐标系

万向坐标系与欧拉XYZ旋转控制器一同使用。它与局部坐标系类似，但其三个旋转轴相互之间不一定垂直，一般在骨骼系统中会使用到

<img src="https://img.picgo.net/2023/06/03/wanxaing9c67f92ad16931ed.png" alt="wanxaing9c67f92ad16931ed.png" border="0"/>

蓝色是z轴，绿色是y轴，红色是x轴

所以 z 是  <span style={{background: '#f3a694'}}>&nbsp;y和x 的父级&nbsp;</span>  ，y是  <span style={{background: '#f3a694'}}>&nbsp;z的父级&nbsp;</span>  ， x 是所有轴的子集

> 所以当改变 **z轴** 的旋转值时，y 和 x 轴都会改变位置

> 当改变 **y轴** 的旋转值时，x 轴会改变位置

就是欧拉角
